##  功能

- 使用vant框架进行下拉刷新  每次刷新随机获取今日头条新闻 
- Tabbar 是用手写的 利用了父子组件的互相传值到达高亮效果
- 文章详情使用Vant框架的选项卡功能
- 首页到文章详情页使用了二级路由传参
- 大麦网数据下的三个分类是三个子路由
- 测试地址：https://xiaohuwei.github.io/appdemo/dist/
